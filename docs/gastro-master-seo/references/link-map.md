## Verlinkungsregeln

### Die 10 Regeln (Kurzfassung)
1. **Max 1x pro Keyword pro Seite** — nur erste sinnvolle Erwähnung verlinken
2. **Nur im Fließtext** — nie in Überschriften, Buttons, Footer, Breadcrumbs, Trust-Bar
3. **Keine Selbstverlinkung** — Seite verlinkt nicht auf sich selbst
4. **Beschreibende Anchor-Texte** — nie "hier klicken" oder nacktes Keyword
5. **Anchors variieren** — kein Anchor >2x auf der gesamten Website für gleiche Zielseite
6. **80/20-Regel** — 80% natürliche Phrasen, 20% keyword-nahe Anchors
7. **Kontextuelle Einbettung** — Link muss sich natürlich in den Satz einfügen
8. **Keine Wettbewerber verlinken** — Namen erwähnen OK (bei Vergleichen), aber nie mit Link
9. **3-5 Links pro 500 Wörter** — nicht übertreiben
10. **Link-Juice-Verteilung** — wichtigste Seiten bekommen die meisten eingehenden Links

### Link-Priorität (absteigend)
1. 🔴 **Hoch:** /produkte/bestellsystem, /produkte/kassensystem, /loesungen/lieferservice-gruenden
2. 🟠 **Mittel-Hoch:** /produkte/bestellapp, /produkte/webseite, /loesungen/restaurant, /loesungen/franchise
3. 🟡 **Mittel:** /produkte/qr-bestellung, /produkte/kiosk-system, /produkte/fahrer-app, /integrationen/*, /wissen/*
4. 🟢 **Niedrig:** /faq, /kontakt, /demo, /ueber-uns, /vergleiche/*

## Kreuzverlinkungs-Matrix

### Cluster-interne Verlinkung (Pflicht)
- Jede /produkte/-Unterseite verlinkt auf mindestens 1 andere /produkte/-Seite
- Jede /loesungen/-Unterseite verlinkt auf mindestens 2 /produkte/-Seiten
- Jede /wissen/-Seite verlinkt auf mindestens 1 /produkte/ + 1 /loesungen/-Seite

### Cluster-übergreifende Verlinkung (empfohlen)
- /produkte/ → /loesungen/ (welche Branche profitiert davon?)
- /loesungen/ → /produkte/ (welche Produkte lösen das Problem?)
- /wissen/ → /produkte/ + /loesungen/ (der Ratgeber führt zur Lösung)
- /faq → alles (FAQ ist der zentrale Content-Hub)
- /vergleiche/ → /produkte/ (nach Vergleich: unser Angebot)
- /integrationen/ → /produkte/ (Integration funktioniert mit diesen Produkten)

### Hub-Seiten verlinken auf Unterseiten
- /produkte/ (Hub) → alle /produkte/[x] Unterseiten
- /loesungen/ (Hub) → alle /loesungen/[x] Unterseiten
- /wissen/ (Hub) → alle /wissen/[x] Unterseiten
